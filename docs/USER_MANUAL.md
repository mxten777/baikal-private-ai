## 1. 시작하기

### 1.1 BAIKAL Private AI란?

BAIKAL Private AI는 **회사 내부 문서를 기반으로 AI가 질문에 답변해주는 시스템**입니다.

- 📄 PDF, Word, Excel 문서를 업로드하면 AI가 내용을 학습합니다
- 💬 자연어로 질문하면 문서 내용을 바탕으로 정확한 답변을 제공합니다
- 🔒 모든 데이터는 회사 내부 서버에서만 처리되어 외부 유출 걱정이 없습니다
- ⚡ 실시간으로 AI가 답변을 생성하는 과정을 볼 수 있습니다

### 1.2 접속 방법

1. 웹 브라우저(Chrome, Edge, Firefox 권장)를 엽니다
2. 주소창에 시스템 관리자가 안내한 주소를 입력합니다
   - 예: `http://서버주소` 또는 `http://192.168.x.x`
3. 로그인 화면이 나타나면 정상입니다

### 1.3 지원 브라우저

| 브라우저 | 지원 여부 |
|----------|:---------:|
| Google Chrome (최신) | ✅ 권장 |
| Microsoft Edge (최신) | ✅ 권장 |
| Mozilla Firefox (최신) | ✅ |
| Safari (최신) | ✅ |
| Internet Explorer | ❌ 미지원 |

## 2. 로그인 / 로그아웃

### 2.1 로그인

1. 접속 화면에서 **사용자명**과 **비밀번호**를 입력합니다
2. **"로그인"** 버튼을 클릭합니다 (또는 Enter 키)
3. 로그인 성공 시 AI 질문응답 화면으로 이동합니다

> **최초 로그인 시**: 시스템 관리자에게 발급받은 계정을 사용합니다. 로그인 후 반드시 비밀번호를 변경하세요.

### 2.2 로그인 실패 시

| 메시지 | 원인 | 해결 방법 |
|--------|------|----------|
| "아이디 또는 비밀번호가 올바르지 않습니다" | 잘못된 입력 | 정확한 아이디/비밀번호 확인 |
| 화면이 표시되지 않음 | 서버 접속 불가 | 네트워크 연결 확인 또는 관리자에게 문의 |

### 2.3 로그아웃

1. 좌측 사이드바 하단의 **자신의 이름** 영역에 마우스를 올립니다
2. 나타나는 **문 아이콘(→)** 을 클릭합니다
3. 로그인 화면으로 돌아갑니다

> **참고**: 브라우저를 닫아도 일정 시간(기본 60분) 내에 다시 접속하면 자동 로그인됩니다. 7일이 지나면 다시 로그인이 필요합니다.

## 4. AI 질문응답

### 4.1 AI 질문응답이란?

업로드된 문서를 바탕으로 AI가 사용자의 질문에 답변하는 핵심 기능입니다. AI는 문서 내용에 기반하여 답변하므로, 관련 문서가 업로드되어 있어야 정확한 답변을 받을 수 있습니다.

### 4.2 새 대화 시작하기

1. 좌측 메뉴에서 **"AI 질문응답"** 을 클릭합니다
2. 채팅 세션 목록 상단의 **"+ 새 대화"** 버튼을 클릭합니다
3. 새 대화 세션이 생성되고 질문을 입력할 수 있습니다

### 4.3 질문하기

1. 화면 하단의 입력창에 **질문을 자연어로 입력**합니다
2. **"전송"** 버튼을 클릭하거나 **Enter** 키를 누릅니다
3. AI가 관련 문서를 자동으로 검색한 후 답변을 생성합니다
4. 답변은 **실시간으로 한 글자씩** 표시됩니다 (스트리밍)

### 4.4 질문 작성 팁

**좋은 질문 예시:**
- ✅ "2025년 인사규정에서 연차 휴가 일수는 어떻게 되나요?"
- ✅ "보안 정책에서 USB 사용에 대한 규정을 알려주세요"
- ✅ "작년 매출 보고서에서 Q4 매출 실적을 요약해주세요"
- ✅ "출장 경비 정산 절차를 단계별로 설명해주세요"

**효과적인 질문을 위한 팁:**
- 구체적인 문서명이나 키워드를 포함하면 더 정확한 답변을 받을 수 있습니다
- 한 번에 하나의 주제에 대해 질문하는 것이 좋습니다
- 후속 질문을 할 수 있습니다 — AI는 이전 대화 맥락을 기억합니다 (최근 5턴)

### 4.5 답변 이해하기

AI 답변에는 다음 정보가 포함됩니다:

- **답변 본문**: 질문에 대한 AI의 답변 (한국어)
- **참고 문서**: 답변 생성에 사용된 문서 목록
  - 문서명
  - 관련 텍스트 일부

> **중요**: AI 답변은 업로드된 문서 내용을 기반으로 합니다. 관련 문서가 없으면 "관련 정보를 찾을 수 없습니다"라고 답변할 수 있습니다.

### 4.6 대화 세션 관리

- **세션 목록**: 좌측에 이전 대화 세션이 시간순으로 표시됩니다
- **세션 전환**: 다른 세션을 클릭하면 이전 대화를 이어볼 수 있습니다
- **세션 삭제**: 세션의 **삭제 버튼(×)** 을 클릭하면 해당 대화가 삭제됩니다
- **대화 컨텍스트**: AI는 같은 세션 내에서 최근 5번의 대화를 기억합니다

### 4.7 AI 응답 중 표시

질문을 전송하면 AI가 답변을 생성하는 동안:
- 하단에 **"AI가 답변을 생성하고 있습니다..."** 인디케이터가 표시됩니다
- 답변이 실시간으로 한 글자씩 나타납니다
- 답변 생성이 완료되면 참고 문서 정보가 표시됩니다

## 5. 문서 관리

### 5.1 문서 관리 페이지

좌측 메뉴의 **"문서 관리"** 를 클릭하면:
- 본인이 업로드한 문서 목록이 표시됩니다
- 각 문서의 파일명, 파일 형식, 크기, 상태, 업로드 날짜를 확인할 수 있습니다

### 5.2 문서 업로드

#### 지원 파일 형식

| 형식 | 확장자 | 최대 크기 |
|------|--------|----------|
| PDF 문서 | .pdf | 100MB |
| Word 문서 | .docx | 100MB |
| Excel 스프레드시트 | .xlsx | 100MB |

> **참고**: .hwp(한글), .ppt(파워포인트), 이미지 파일은 현재 지원하지 않습니다.

#### 업로드 방법

**방법 1: 드래그 앤 드롭**
1. 문서 관리 페이지의 **업로드 영역**(점선 테두리)을 찾습니다
2. 파일 탐색기에서 업로드할 파일을 **끌어서** 업로드 영역에 **놓습니다**

**방법 2: 파일 선택**
1. 업로드 영역을 **클릭**합니다
2. 파일 선택 대화상자에서 원하는 파일을 선택합니다
3. **"열기"** 를 클릭합니다

#### 업로드 후 처리 과정

업로드한 문서는 자동으로 다음 과정을 거칩니다:

```
업로드 중 → 처리 중 → 완료
  (파일 전송)  (텍스트 추출 + AI 분석)  (질문응답 가능)
```

| 상태 | 의미 | 소요 시간 |
|------|------|----------|
| **업로드 중** (파랑) | 파일이 서버로 전송 중 | 파일 크기에 따라 다름 |
| **처리 중** (노랑) | 텍스트 추출 및 AI 분석 중 | 30초 ~ 수 분 |
| **완료** (초록) | AI 질문응답 가능 | - |
| **실패** (빨강) | 처리 중 오류 발생 | - |

> **팁**: 문서 처리가 완료되면 즉시 AI 질문응답에서 해당 문서 내용에 대해 질문할 수 있습니다.

### 5.3 문서 다운로드

1. 문서 목록에서 다운로드할 문서를 찾습니다
2. **다운로드 버튼(⬇)** 을 클릭합니다
3. 원본 파일이 PC에 저장됩니다

### 5.4 문서 처리 실패 시

문서 상태가 **"실패"** 로 표시되면:
- 에러 메시지를 확인합니다
- 파일이 손상되지 않았는지 확인합니다 (PC에서 정상적으로 열리는지)
- 파일을 다시 업로드해 봅니다
- 문제가 지속되면 시스템 관리자에게 문의합니다

## 6. 문서 검색

### 6.1 문서 검색이란?

업로드된 전체 문서에서 키워드 또는 의미 기반으로 관련 내용을 검색하는 기능입니다. AI 질문응답과 달리, 문서의 원본 텍스트 조각을 직접 찾아볼 수 있습니다.

### 6.2 검색 방법

1. 좌측 메뉴에서 **"문서 검색"** 을 클릭합니다
2. 검색창에 **키워드 또는 문장을 입력**합니다
3. **"검색"** 버튼을 클릭하거나 **Enter** 키를 누릅니다

### 6.3 검색 모드

| 모드 | 설명 | 적합한 경우 |
|------|------|------------|
| **하이브리드** (기본) | 키워드 + 의미 검색 결합 | 일반적인 검색 (권장) |
| **키워드** | 정확한 단어 일치 검색 | 특정 용어, 코드, 번호 검색 |
| **벡터** | 의미 유사도 기반 검색 | 비슷한 개념, 관련 내용 검색 |

### 6.4 검색 결과 이해하기

검색 결과에는 각 항목에 다음 정보가 표시됩니다:

- **유사도 점수** (0~100%): 검색어와의 관련도
- **문서명**: 원본 파일명
- **텍스트 내용**: 검색어와 관련된 텍스트 조각
- **문서 유형**: PDF / DOCX / XLSX

### 6.5 검색 팁

- **구체적인 키워드**: "휴가" 보다 "연차 휴가 일수"가 더 정확한 결과를 반환합니다
- **자연어 검색**: "출장 가면 교통비는 어떻게 정산하나요?" 같은 질문형도 가능합니다
- **하이브리드 모드**: 대부분의 경우 가장 좋은 결과를 제공합니다

## 7. 비밀번호 변경

### 7.1 비밀번호 변경 방법

1. 좌측 사이드바 **하단**의 사용자 프로필 영역에 **마우스를 올립니다**
2. 나타나는 **열쇠 아이콘(🔑)** 을 클릭합니다
3. 비밀번호 변경 팝업이 열립니다

### 7.2 비밀번호 변경 절차

1. **현재 비밀번호**: 지금 사용 중인 비밀번호를 입력합니다
2. **새 비밀번호**: 변경할 비밀번호를 입력합니다 (4자 이상)
3. **비밀번호 확인**: 새 비밀번호를 한 번 더 입력합니다
4. **"비밀번호 변경"** 버튼을 클릭합니다

### 7.3 비밀번호 규칙

- **최소 4자 이상**이어야 합니다
- 새 비밀번호와 비밀번호 확인이 일치해야 합니다
- 현재 비밀번호가 올바르지 않으면 변경되지 않습니다

### 7.4 비밀번호 변경 성공

- "비밀번호가 변경되었습니다" 메시지가 표시됩니다
- 팝업이 자동으로 닫힙니다
- 다음 로그인부터 새 비밀번호를 사용합니다
- 현재 세션은 유지됩니다 (재로그인 불필요)

### 7.5 비밀번호를 잊어버렸을 때

비밀번호를 잊어버린 경우 **시스템 관리자에게 문의**하여 비밀번호 초기화를 요청하세요.

## 8. 모바일 사용법

BAIKAL Private AI는 모바일 기기(스마트폰, 태블릿)에서도 사용할 수 있습니다.

### 8.1 모바일 화면 구성

모바일에서는 사이드바가 숨겨지고, 상단에 도구 모음이 표시됩니다:

```
┌─────────────────────────┐
│ ☰  BAIKAL               │  ← 상단 바
├─────────────────────────┤
│                         │
│     메인 콘텐츠          │
│                         │
│                         │
│                         │
└─────────────────────────┘
```

### 8.2 메뉴 열기/닫기

1. 좌측 상단의 **≡ (햄버거) 아이콘**을 탭합니다
2. 사이드바 메뉴가 왼쪽에서 슬라이드되어 나타납니다
3. 메뉴 항목을 탭하여 페이지를 이동합니다
4. 메뉴 바깥 영역을 탭하거나 뒤로가기를 누르면 메뉴가 닫힙니다

### 8.3 모바일에서 AI 질문응답

- 채팅 세션 목록은 별도의 토글 버튼으로 열고 닫을 수 있습니다
- 질문 입력 후 전송 버튼을 탭합니다
- 답변은 화면 전체 너비로 표시됩니다

### 8.4 모바일에서 문서 업로드

- 업로드 영역을 탭하면 파일 선택이 열립니다
- 기기에 저장된 PDF/DOCX/XLSX 파일을 선택할 수 있습니다
- 문서 목록은 세로로 스크롤됩니다

## 9. 자주 묻는 질문 (FAQ)

### Q1. AI가 "관련 문서를 찾을 수 없습니다"라고 답변합니다

**A:** 질문과 관련된 문서가 아직 업로드되지 않았거나, 문서 처리가 완료되지 않았을 수 있습니다.
- 문서 관리 페이지에서 관련 문서의 상태가 **"완료"** 인지 확인하세요
- 문서가 없다면 관련 문서를 업로드한 후 다시 질문하세요

### Q2. AI 답변이 느립니다

**A:** AI 답변 생성 속도는 서버 성능에 따라 달라집니다.
- CPU만 사용하는 서버에서는 10~30초 정도 걸릴 수 있습니다
- 긴 질문이나 복잡한 질문일수록 시간이 더 걸립니다
- 잠시 기다려 주세요 — 답변이 실시간으로 표시됩니다

### Q3. 업로드한 문서를 삭제하고 싶습니다

**A:** 일반 사용자는 직접 삭제할 수 없습니다. 시스템 관리자에게 삭제를 요청하세요. 관리자만 문서를 삭제할 수 있습니다.

### Q4. 이전 대화를 다시 볼 수 있나요?

**A:** 네, AI 질문응답 화면의 좌측 세션 목록에서 이전 대화 세션을 클릭하면 대화 기록을 확인할 수 있습니다.

### Q5. 같은 문서를 여러 번 업로드해도 되나요?

**A:** 가능하지만, 같은 문서가 중복으로 처리되어 AI 답변에 중복 내용이 포함될 수 있습니다. 가능하면 동일 문서의 중복 업로드는 피해주세요.

### Q6. PPT나 HWP 파일도 업로드할 수 있나요?

**A:** 현재는 PDF(.pdf), Word(.docx), Excel(.xlsx) 파일만 지원합니다. HWP나 PPT 파일은 PDF로 변환한 후 업로드해주세요.

### Q7. AI 답변이 정확하지 않습니다

**A:** AI 답변은 업로드된 문서 내용을 기반으로 생성됩니다.
- 질문을 더 구체적으로 작성해 보세요
- 관련 문서가 모두 업로드되었는지 확인하세요
- 여전히 부정확하면 직접 문서를 확인하거나 담당자에게 문의하세요

### Q8. 세션을 삭제하면 복구할 수 있나요?

**A:** 아니요, 삭제된 대화 세션은 복구할 수 없습니다. 삭제 전에 필요한 답변 내용을 복사해두세요.

### Q9. 여러 문서에 걸쳐서 질문할 수 있나요?

**A:** 네, AI는 업로드된 모든 문서에서 관련 내용을 검색합니다. "인사규정과 복무규정에서 휴가에 대한 내용을 비교해줘"와 같은 질문도 가능합니다.

### Q10. 다른 사람이 업로드한 문서도 검색되나요?

**A:** 네, AI 질문응답과 문서 검색은 시스템에 등록된 모든 문서를 대상으로 합니다. 단, 문서 관리 목록에서는 관리자를 제외하고 본인이 업로드한 문서만 보입니다.

## 10. 문제 해결

### 10.1 화면이 로딩되지 않을 때

1. 인터넷 (또는 사내망) 연결을 확인합니다
2. 브라우저를 새로고침합니다 (**Ctrl + F5** 또는 **Cmd + Shift + R**)
3. 브라우저 캐시를 삭제합니다
4. 다른 브라우저로 시도합니다
5. 문제가 지속되면 관리자에게 문의합니다

### 10.2 로그인이 갑자기 풀렸을 때

- 일정 시간(기본 60분) 동안 사용하지 않으면 세션이 만료됩니다
- 다시 로그인하면 이전 대화 기록은 모두 유지되어 있습니다

### 10.3 "서버에 연결할 수 없습니다" 메시지

- 서버가 일시적으로 점검 중일 수 있습니다
- 잠시 후 다시 시도합니다
- 문제가 지속되면 관리자에게 서버 상태를 확인 요청합니다

### 10.4 "AI 서비스가 사용할 수 없습니다" 메시지

- AI 엔진이 과부하 상태일 수 있습니다
- 잠시(30초~1분) 기다린 후 다시 질문합니다
- 다른 사용자가 동시에 많은 질문을 하고 있을 수 있습니다

### 10.5 문서 업로드가 진행되지 않을 때

1. 파일 형식이 PDF/DOCX/XLSX인지 확인합니다
2. 파일 크기가 100MB를 초과하지 않는지 확인합니다
3. 파일이 PC에서 정상적으로 열리는지 확인합니다
4. 페이지를 새로고침한 후 다시 시도합니다
